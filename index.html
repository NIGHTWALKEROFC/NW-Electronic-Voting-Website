Demo mode simulates random voting activity so you can test how the site looks with votes. All demo votes are clearly marked and do not affect real results. Admin can stop demo mode at any time — demo votes are then cleared.

ℹ Demo votes are wiped when demo is stopped. Real votes are never touched by demo mode.
